The official implementation for ICML24 paper "Graph Out-of-Distribution Detection Goes Neighborhood Shaping"

## Dependencies
- python == 3.8.18
- torch == 1.13.0
- torch-geometric == 2.3.1


## How to run the code

For TopoOOD, please run

```bash
$ pip install -r requirement.txt
$ bash run_topoood.sh
```
